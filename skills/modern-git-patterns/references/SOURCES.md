1. Edward Thomson, "You Don't Know Git", NDC London 2025

   Focus: Git internals, libgit2, making git more usable

2. "you only need 15ish git commands"
   https://youtu.be/y6meVhckYmQ
   Focus: Essential git command set, practical workflow

3. "Level Up Your Git Game: 10 Little-Known Features You'll Love"
   https://www.youtube.com/watch?v=bVnsBnnW3Tw
   Focus: git notes, worktree, .gitkeep, reflog, format-patch,
   request-pull, bisect, rev-list, hooks

4. "So You Think You Know Git Part 2", DevWorld 2024
   https://www.youtube.com/watch?v=Md44rcw13k4
   Focus: Advanced git patterns, git internals deep-dive

5. Scott Chacon, "So You Think You Know Git", FOSDEM 2024
   https://www.youtube.com/watch?v=aolI_Rz0ZqY

   prefetching, column UI, includeIf, word-diff, range-diff,
   GitButler virtual branches

6. "Why everyone hates git submodules"
   https://www.youtube.com/watch?v=JESI498HSMA
   Focus: Submodule problems, alternatives (package managers, monorepo)
