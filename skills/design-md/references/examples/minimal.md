# Minimal DESIGN.md Example

Smallest valid DESIGN.md file. Use as a starting template.

```markdown
---
name: My Project
colors:
  primary: "#1A1C1E"
  neutral: "#FFFFFF"
typography:
  body-md:
    fontFamily: system-ui
    fontSize: 16px
    fontWeight: 400
    lineHeight: 1.6
---

## Overview

Clean and minimal design system. Dark text on light backgrounds with system fonts for maximum readability and performance.

## Colors

- **Primary (#1A1C1E):** Used for all text and primary interactive elements.
- **Neutral (#FFFFFF):** Background and surface color.

## Typography

System font stack for optimal loading performance. Single body type scale at 16px with comfortable 1.6 line height.
```
