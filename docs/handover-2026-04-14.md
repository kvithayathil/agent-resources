### Architecture

```
security_layer/
├── PURE CORE (deterministic, fuzz-tested):
│   ├── budget.py            — rate limiting, token budgets
│   ├── output_validation.py — injection detection, schema validation
│   ├── models.py            — frozen dataclasses (ContentClassification, TaintFlag, etc.)
│   ├── state_machine.py     — generic StateMachine[S,E,C]
│   ├── permissions.py       — path traversal, capability escalation
│   ├── trifecta.py          — lethal trifecta violation detection
│   └── scanning.py          — CUSTOM_RULES stage (regex)
├── EFFECTFUL SHELL:
│   ├── pii.py               — Presidio-based PII detection/redaction
│   ├── scanning.py          — SEMGREP stage, secret scanning

├── classifier.py            — input classification
└── incident.py              — incident response state machine

skills/scripts/
└── sync-index.py            — skill index manager (sync, validate, search, lookup, tag, tokens)
```
